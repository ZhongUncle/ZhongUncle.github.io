  • Verify the central limit theorem using C++ (by generating a PPM format image to verify that the discrete event with non-uniform probability conforms a normal distribution curve when the sample size is large enough)

    2023-05-16 ｜ #Words: 2545

    This blog explains how to use C++ to verify the central limit theorem (CLT), which states that the sum of independent random variables tends to follow a normal distribution when the sample size is large enough. ZhongUncle uses dice points as an example of discrete events, and generates PPM format images to visualize the probability distribution of the sum of points. ZhongUncle shows that the distribution curve becomes more similar to a normal curve as the number of samples increases, and that this holds true even when the probability of each dice point is not uniform. ZhongUncle provides the source code and the output images for different scenarios, and invites the readers to try it themselves. ZhongUncle also expresses the fascination with the central limit theorem and the normal distribution.

    Typedef 是什么？

    typedef是 C 语言中的一个特色功能，被用于创建新的类型名称，从名称“type def(ine)”也可以看出其功能。typedef类似于 Unix 中的alias程序的功能，为一个对象添加另外一个名称，typedef使得多个名称其实对应的是同一个程序，但是typedef操作的对象是数据类型，也就是给数据类型“起外号”，这个外号一般首字母大写，用于表示不是基础数据类型。
