    Typedef 是什么？ typedef是 C 语言中的一个特色功能，被用于创建新的类型名称，从名称“type def(ine)”也可以看出其功能。typedef类似于 Unix 中的alias程序的功能，为一个对象添加另外一个名称，typedef使得多个名称其实对应的是同一个程序，但是typedef操作的对象是数据类型，也就是给数据类型“起外号”，这个外号一般首字母大写，用于表示不是基础数据类型。
