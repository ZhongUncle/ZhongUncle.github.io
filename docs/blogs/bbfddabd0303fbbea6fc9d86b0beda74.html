Windows 如何安装 WSL（中途退出后遇到 bug 该怎么办）

2022-07-10 ｜ UNIX ｜ #Words: 441

适用于 Linux 的 Windows 子系统，被简称为WSL，安装方式很简单。但是 WSL 还是不如原生的 Linux，只能某些时候临时用一下。

如果你是第一次安装，打开Windows PowerShell（推荐使用Windows Termianl，可以使用商店下载安装），输入以下命令会自动安装Ubuntu：

wsl --install

但是如果你觉得太慢关掉了，或者出现问题退出了，或者想安装其他的Linux发行版的话，那么就需要使用以下方式。 首先使用以下命令查看可安装的Linux发行版列表：

wsl --list --online

结果如下：

以下是可安装的有效分发的列表。
请使用“wsl --install -d <分发>”安装。

NAME            FRIENDLY NAME
Ubuntu          Ubuntu
Debian          Debian GNU/Linux
kali-linux      Kali Linux Rolling
openSUSE-42     openSUSE Leap 42
SLES-12         SUSE Linux Enterprise Server v12
Ubuntu-16.04    Ubuntu 16.04 LTS
Ubuntu-18.04    Ubuntu 18.04 LTS
Ubuntu-20.04    Ubuntu 20.04 LTS

需要注意，这里的分发名称是NAME这列写的，例如Ubuntu-20.04

那么我们输入以下命令安装Ubuntu-20.04：

wsl --install -d Ubuntu-20.04

然后就会显示

正在下载: Ubuntu
[=========                 15.9%                           ]

这时候就在安装啦，安心等待就好。

如果出现以下问题，如下：

安装过程中出现错误。分发名称: 'Ubuntu' 错误代码: 0x80072ee2]

就需要手动重新开始一下。 如果还有问题，请参考WSL安装过程中出现 0x80072EE2 错误。
