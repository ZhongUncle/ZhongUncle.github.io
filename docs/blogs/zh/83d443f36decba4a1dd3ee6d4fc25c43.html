有时候为了学习或者调试，需要查看程序的汇编代码。这在很多 GUI 的 IDE 中很方便实现，例如在 Xcode 中，点击“Product”-“Perform Action”-“Assemble 文件名”就可以，如下：

那么在终端中呢？

首先写一个 C 源码文件 test.c。内容如下：

#include <stdio.h>

int main() {
    printf("hello world!\n");
    return 0;
}

然后使用以下命令：

$ cc -S test.c

这时候会产生一个名为test.s的文件，里面就是汇编语言，如下：
